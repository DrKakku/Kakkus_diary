# 🕷️ Spider Exercise

> [!info] What this is
> The spider exercise is a fretting-hand coordination drill. It trains finger independence, hand synchronization, clean fretting, and relaxed movement.

## Purpose
- Finger independence
- Left-hand accuracy
- Right-hand and left-hand synchronization
- Alternate picking control
- Relaxed movement under a metronome

## How to practice
1. Start very slow.
2. Use one finger per fret.
3. Keep unused fingers close to the strings.
4. Pick with strict alternate picking.
5. Stop if tension rises.
6. Increase tempo only after several clean runs.

> [!tip] Form rules
> - Press only as hard as needed
> - Keep the thumb relaxed
> - Minimize finger lift
> - Prioritize cleanliness over speed

> [!warning]
> If the hand starts to tense up, reduce the tempo immediately.

## Core pattern

    e|-------------------------1-2-3-4-|
    B|-----------------1-2-3-4---------|
    G|---------1-2-3-4------------------|
    D|-1-2-3-4--------------------------|
    A|----------------------------------|
    E|----------------------------------|

## Common variations
- Standard 1-2-3-4
- Reverse 4-3-2-1
- Ascend and descend
- Different string sets
- One-string version
- String skipping
- Finger permutation changes

## Tempo guide

| Stage | BPM range | Focus |
|---|---:|---|
| Warm-up | 40–60 | Relaxation and clean notes |
| Control | 60–90 | Even picking and finger accuracy |
| Build | 90–120 | Consistency under tempo |
| Push | 120+ | Only if form stays clean |
