# 🎸 CAGED System + Basic Music Theory

---

## 🧠 1. What is a chord (basic theory)

### ✅ A chord = multiple notes played together

The most common type:

> **Major chord = 1 (root) + 3 + 5**

This comes from a **major scale**

---

### 🎼 Example: C Major Scale

```
C  D  E  F  G  A  B
1  2  3  4  5  6  7
```

👉 So:

- **C major chord = C (1) + E (3) + G (5)**
    

---

### 🔑 Formula (important)

|Chord Type|Formula|Example (C)|
|---|---|---|
|Major|1–3–5|C–E–G|
|Minor|1–♭3–5|C–Eb–G|

---

### 🎯 What this means for guitar

Every chord shape you play is just:

- A **rearrangement of 1–3–5**
    
- Spread across strings
    

---

## 🎸 2. What is the CAGED System

The **CAGED system** is a way to map the fretboard using 5 chord shapes:

```
C → A → G → E → D → (repeat)
```

Each shape:

- Is based on an **open chord**
    
- Can be moved to form **any chord**
    
- Contains chord tones (1–3–5)
    

---

## 🧩 3. The 5 CAGED Shapes

---

### 🔹 E Shape (Start here)

- Based on open E major
    
- Fully movable (barre chord)
    

**Roots:**

- Low E string
    
- High E string
    

**Example:**

- 1st fret → F
    
- 3rd fret → G
    

---

### 🔹 A Shape

- Based on open A major
    
- Barre across strings
    

**Root:**

- A string
    

**Example:**

- 3rd fret → C
    
- 5th fret → D
    

---

### 🔹 D Shape

- Smaller shape
    
- Higher pitch
    

**Root:**

- D string
    

**Use:**

- Lead playing
    
- Triads
    

---

### 🔹 C Shape

- More complex
    
- Not commonly used fully
    

**Use:**

- Partial chords
    

---

### 🔹 G Shape

- Physically awkward
    
- Rarely used fully
    

**Use:**

- Break into smaller parts
    

---

## 🔗 4. The Big Idea (very important)

Each chord exists in **5 positions across the neck**

Example: C major

- C shape → open position
    
- A shape → 3rd fret
    
- G shape → 5th fret
    
- E shape → 8th fret
    
- D shape → 10th fret
    

👉 These are all the **same chord**

---

## 🧠 5. Root Notes (critical concept)

The **root note defines the chord**

If you know:

- Where the root is
    
- What shape you're using
    

👉 You can find **any chord instantly**

---

### 🎯 Quick reference

|Shape|Root location|
|---|---|
|E|Low + high E|
|A|A string|
|D|D string|
|C|A/B region|
|G|Low + high E|

---

## 🎼 6. How CAGED connects to theory

Each shape contains:

- Chord tones (1–3–5)
    
- Nearby scale notes
    

👉 This means:

- You can **build chords**
    
- You can **solo around them**
    

---

## 🧪 7. Practice Routine (structured)

---

### 🔹 Step 1: Learn E + A shapes (priority)

- Play:
    
    - E shape → move it up the neck
        
    - A shape → move it up
        

Focus:

- Clean sound
    
- Root note awareness
    

---

### 🔹 Step 2: Pick ONE key (important)

Example: **C major**

Find C in:

- A shape (3rd fret)
    
- E shape (8th fret)
    

---

### 🔹 Step 3: Connect shapes

- Move between shapes
    
- Don’t think in “boxes”
    
- Think in **overlapping zones**
    

---

### 🔹 Step 4: Add basic scale awareness

Inside each shape:

- Locate:
    
    - Root (1)
        
    - Third (3)
        
    - Fifth (5)
        

---

### 🔹 Step 5: Apply to songs

Example progression:

```
C – G – Am – F
```

Try:

- Playing in one position
    
- Then shifting using CAGED
    

---

## 🔁 8. Minimal Daily Practice (10–15 min)

- 3 min → Chromatic warmup
    
- 5 min → E + A shapes
    
- 3 min → One key (map positions)
    
- 2 min → Simple progression
    

---

## ⚠️ Common Mistakes

- Memorizing shapes without understanding **1–3–5**
    
- Ignoring root notes
    
- Trying all 5 shapes at once
    
- Not applying to real music
    

---

## 🧠 Mental Model (keep this simple)

> CAGED =  
> **Chord shape + root note + (1–3–5 awareness)**  
> = full fretboard understanding
